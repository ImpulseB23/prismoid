One chat feed. Every platform. Built for streamers who refuse to choose.

What it does
Unified feed
Messages from all platforms flow into a single stream. No tab switching.
Cross-platform mod
Ban, timeout, and delete from one interface regardless of source platform.
Universal emotes
7TV, BTTV, FFZ render everywhere. Even in YouTube chat.
OBS overlay
Browser source URL that renders clean unified chat. No platform badges visible to viewers.
